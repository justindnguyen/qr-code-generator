# QR Code Generator

A simple web app to create QR codes from text or links.

## Features

- Generate QR codes instantly
- Download QR code as an image
- Works on desktop and mobile

## How to Run

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the server:

   ```bash
   npm start
   ```

The app will open at `http://localhost:3002`.

## Built With

- HTML
- CSS
- JavaScript
- QRCode.js
